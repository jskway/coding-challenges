# Queues

A **queue** stores items in a first-in, first-out (FIFO) order.
First come, first served.

### Strengths

- **Fast operations.** All queue operations take _O(1)_ time.

|               | Worst Case |
| ------------- | ---------- |
| **_space_**   | _O(n)_     |
| **_enqueue_** | _O(1)_     |
| **_dequeue_** | _O(1)_     |
| **_peek_**    | _O(1)_     |
